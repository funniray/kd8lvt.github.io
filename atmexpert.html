"All The Mods Expert" Installation Instructions

PLEASE NOTE THAT THERE IS A RECOMMENDED AMOUNT OF RAM YOU SHOULD HAVE ON YOUR COMPUTER AT MINIMUM. THIS NUMBER IS SIX GIGABYTES.

PLEASE MAKE SURE YOU HAVE AT LEAST SIX GIGABYTES OF RAM FIRST! Here's how:

How much ram do I have?
  1. Start by installing Curse Voice, if you haven't already. I'm assuming you already have Minecraft installed. If not, install that first.
  2. Launch Curse Voice (What, you expected me to make you install something but not launch it right away?)
  3. Log in with a Curse account. If you have a Minecraft Forums acount already, use that. Otherwise, create a new one.
  4. Next, click on the creeper face on the left-hand bar. It should be all the way at the bottom.
  5. Near the top you should see a "Browse All Modpacks" button. Click that and type in the search bar (it should be on the left if you can't find it) "All The Mods Expert"
  6. Find All the Mods Expert in the list (it should be the only one, in theory)
  7. Click the image. DO NOT CLICK INSTALL! That will install the latest version, which the server is not updated to.
  8. Click where it says "Versions"
  9. Click the download button in the list that has "allthemods Expert v1.0.21" in the middle of it's row. The download buttons are all on the right hand side.
  10. Wait a while. Depending on your internet speed this can take anywhere from a few seconds to a few minutes. Be patient!
  11. Once ATMExpert has finished downloading, click the settings wheel on the left-hand side of the window we've been using all the way at the bottom. (That's Curse.)
  12. Click the Creeper face again, this time in the settings window though. It's the third one from the bottom on mine.
  13. Scroll down until you either see "Advanced Settings" or "Launcher Settings"
  14. If you see "Advanced Settings", click it. If you see "Launcher Settings", go to the next step.
  15. Scroll down until you see "JAVA SETTINGS" and then "MAXIMUM MEMORY ALLOCATION" Slide the bar up until it says "4,096" above it.
  16. Save the settings, and then go back to the Minecraft menu.
  17. Launch All The Mods Expert by hovering over the profile with your mouse, and clicking "Play"
  18. Wait for it to launch. Depending on your computer, this can take anywhere from a few minutes, to an hour. For example, I have a pretty bad computer, so it takes about a half-hour. My brother has a great computer, so it takes him about fifteem minutes.
  19. Once it has launched put the normal BASCraft ip in. If you do not know it, and can prove you go to Belding Schools, friend me on Curse (kd8lvt) and I can give you the IP.

Congratulations! You can now join the server, and play! If you need help with it (which you will, trust me.) Craft one dirt by itself to get a book that will teach you the basics (don't ask, I have no idea how it works either.)
